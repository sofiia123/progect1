Списки

Види сайтів:

  1. Статичні - це сайти, що не мають back-end, тобто не мають бази з даними. Вони створюються через :
    • HTML
    • CSS
    • інколи JS...
  2. Динамічні - це сайти, що мають back-end, тобто беруть дані з бази. Вони створюються через:
    • HTML
    • CSS
    • обов'язково JS
